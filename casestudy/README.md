## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Navigate to the project directory:
```bash
cd casestudy
```

2. Install dependencies:
```bash
npm install
```

### Running the Application

Start the development server:
```bash
npm start
```

The application will open in your browser at [http://localhost:3000](http://localhost:3000).

### Building for Production

Create an optimized production build:
```bash
npm run build
```

## Features Overview

### Landing Page
- Eye-catching hero section with live space feed
- Partner organizations showcase
- Call-to-action buttons for mission enrollment

### Authentication
- Secure login interface for telescope access
- Mission control-themed design

### Mission Catalog
- Browse available observation missions
- Search functionality by title or category
- Mission cards with ratings, participants, and pricing
- One-click reservation system

### Active Missions
- Track enrolled observation missions
- Progress indicators for data collection
- Quick access to lens views

### Feedback System
- Star-based rating system
- Mission-specific feedback forms
- Submission confirmation with animations
