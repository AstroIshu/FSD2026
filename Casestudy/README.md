# 🌌 Lumina Space — Deep Space Research Platform

**Lumina Space** is a futuristic, single-page React application that simulates a deep-space research and observation platform. It allows users to explore celestial research missions, reserve observations, track active missions, and submit mission debriefs — all wrapped in an olive-themed, deep-space UI.

Designed with a **research-grade aesthetic**, **responsive UX**, and **mission-driven storytelling**, this project showcases modern React patterns and polished UI engineering.

---

## ✨ Features

### 🚀 Landing Experience
- Immersive hero section with live cosmic feed
- Research-oriented branding and mission narrative
- Responsive layout with mobile navigation

### 🔐 Authentication (Mock)
- Mission Control login flow
- Session-based user state (no backend required)

### 🪐 Observation Catalog
- Browse curated deep-space missions
- Filter missions via real-time search
- Mission metadata: rating, duration, category, price, scientist

### 📡 Mission Reservation
- Reserve missions with one click
- Track enrolled missions separately
- Visual mission status and progress indicators

### 📊 Active Missions Dashboard
- Progress visualization
- Mission status badges
- Empty-state UX when no missions are active

### 📝 Mission Debrief / Feedback
- Star-based rating UI
- Mission-specific feedback
- Submission confirmation state

### 📱 Fully Responsive
- Desktop, tablet, and mobile-optimized
- Animated mobile navigation menu

---

## 🧠 Tech Stack

- **React** (Hooks-based architecture)
- **Tailwind CSS** (utility-first styling)
- **Lucide Icons** (modern SVG icon set)
- **Pure Frontend** (no backend / API calls)
- **Single-file App Architecture**
